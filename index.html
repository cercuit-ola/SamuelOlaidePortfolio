Fullstack JavaScript and cloud engineer specializing in React, TypeScript, Node.js (Express) and Python backend systems, with production experience across AWS, Google Cloud (GCP) and Azure. I build scalable, secure distributed systems, CI/CD pipelines, containerized microservices and modern DevOps infrastructure for cloud-native and AI-driven platforms. Alongside software engineering, I conduct geospatial and remote-sensing research using satellite gravimetry (GRACE / GLDAS), GIS and geophysical data to analyze groundwater systems, lithospheric structure and resource distribution; connecting software, Earth observation and data science into real-world analytical and decision-support systems.

Measuring Nigeria's aquifers from orbit.

The work pairs an interactive data visualiser with peer-reviewed science, rooted in a physics background and an environmental-data research practice.
